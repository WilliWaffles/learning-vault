# Learning Vault

Desktop application for tracking programming learning, exercises, notes and review sessions.

# Problem to solve

Programming is an extensive field. Each time we learn something new, previous topics can get buried under an enormous pile of information. Learning Vault serves as a place to save, organize, and review this knowledge.

It helps users keep proof of what they have learned and revisit it whenever they need it.

# MVP v0.1

Language: Python

Topics:
- Functions
- Conditionals
- Loops
- Exceptions

Features:
- Show topics and their respective theory
- Questions and quizzes
- Exercises for topics
- Mark exercise as Completed
- Save everything on SQLite

# Future Features

- Add more languages
- More concepts (data types, OOP, Git, Big O, Algorithms)
- Different type of quizzes with AI responses and validation
- Exercises with difficulties, input/output, constraints, hints
- Testing scripts to run in an IDE

# Initial Stack

- Python
- Tkinter
- SQLite
- Git/GitHub
- Markdown for docs

# Non-Goals for v0.1

- No code execution inside the app
- No user accounts
- No cloud sync
- No AI validation
- No advanced statistics
